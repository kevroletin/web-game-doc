###getMessages
#####Format:
    {
      "action": "getMessages",
      "since": <since>
    }
#####Success:
    {"result": "ok", "messages": messages}
      
#####Fail:
    {"result": "badJson"}
      
#####Description:
Позвращает последие 100 собщений, отосланных не позднее, чем <time>

**messages** - список объёктов в формате `{"userId": userId, "text":
  text, "time": time}`
